rule_id: L016
validator: opa
description: pause without seconds/minutes prompts for input.
---

## No prompting

pause without seconds/minutes prompts for input.

### Example: violation

```yaml
- name: Example play
  hosts: localhost
  connection: local
  tasks:
    - name: Bad
      ansible.builtin.shell: whoami
```

### Example: pass

```yaml
- name: Example play
  hosts: localhost
  connection: local
  tasks:
    - name: Ok
      ansible.builtin.command: whoami
```
