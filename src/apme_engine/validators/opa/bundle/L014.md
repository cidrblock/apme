---
rule_id: L014
validator: opa
description: Use notify/handler instead of when: result.changed.
---

## No handler

Use notify/handler instead of when: result.changed.

### Example: violation

```yaml
- name: Example play
  hosts: localhost
  connection: local
  tasks:
    - name: Bad
      ansible.builtin.shell: whoami
```

### Example: pass

```yaml
- name: Example play
  hosts: localhost
  connection: local
  tasks:
    - name: Ok
      ansible.builtin.command: whoami
```
