## paramiko_ssh connection plugin (M018)

paramiko_ssh connection plugin is deprecated; use ssh connection instead (removed in 2.21)

**Removal version**: 2.21
**Fix tier**: 1
**Audience**: content

### Detection

Detect connection: paramiko_ssh or ansible_connection: paramiko_ssh

Detects `connection: paramiko_ssh` on plays and `ansible_connection: paramiko_ssh`
in task vars. Play-level connection attributes and task vars are represented
differently in the OPA hierarchy; doc examples cannot be tested in the harness.
OPA unit tests in `M018_test.rego` validate the Rego logic directly.

### Remediation

Direct substitution
