rule_id: L010
validator: opa
description: Use failed_when or register instead of ignore_errors.
---

## Ignore errors

Use failed_when or register instead of ignore_errors.

### Example: violation

```yaml
- name: Example play
  hosts: localhost
  connection: local
  tasks:
    - name: Bad
      ansible.builtin.shell: whoami
```

### Example: pass

```yaml
- name: Example play
  hosts: localhost
  connection: local
  tasks:
    - name: Ok
      ansible.builtin.command: whoami
```
