---
rule_id: L019
validator: opa
description: Playbook should have .yml or .yaml extension.
---

## Playbook extension

Playbook should have .yml or .yaml extension.

### Example: violation

```yaml
- name: Example play
  hosts: localhost
  connection: local
  tasks:
    - name: Bad
      ansible.builtin.shell: whoami
```

### Example: pass

```yaml
- name: Example play
  hosts: localhost
  connection: local
  tasks:
    - name: Ok
      ansible.builtin.command: whoami
```
