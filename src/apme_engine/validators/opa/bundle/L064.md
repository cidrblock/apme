## Avoid end_play (L064)

Avoid `meta: end_play` which stops the entire play for all hosts. Prefer
`meta: end_host` to stop only the current host. The `meta` module is not
represented as a standard taskcall node in the content graph, so these examples
cannot be tested through the integration harness. OPA unit tests in
`L064_test.rego` validate the Rego logic directly.

### Example: violation

```yaml
- name: Stop if not Debian
  ansible.builtin.meta: end_play
```

### Example: pass

```yaml
- name: Stop if not Debian
  ansible.builtin.meta: end_host
```
