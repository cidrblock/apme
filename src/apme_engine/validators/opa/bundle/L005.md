rule_id: L005
validator: opa
description: Use ansible.builtin or ansible.legacy.
---

## Only builtins

Use ansible.builtin or ansible.legacy.

### Example: violation

```yaml
- name: Example play
  hosts: localhost
  connection: local
  tasks:
    - name: Bad
      ansible.builtin.shell: whoami
```

### Example: pass

```yaml
- name: Example play
  hosts: localhost
  connection: local
  tasks:
    - name: Ok
      ansible.builtin.command: whoami
```
