rule_id: L022
validator: opa
description: Shell with pipe should set set -o pipefail.
---

## Risky shell pipe

Shell with pipe should set set -o pipefail.

### Example: violation

```yaml
- name: Example play
  hosts: localhost
  connection: local
  tasks:
    - name: Bad
      ansible.builtin.shell: whoami
```

### Example: pass

```yaml
- name: Example play
  hosts: localhost
  connection: local
  tasks:
    - name: Ok
      ansible.builtin.command: whoami
```
