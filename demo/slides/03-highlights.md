
# What Does APME Catch?

  **L** — Lint: FQCN, modes, task names, free-form, bare vars
  **M** — Modernize: deprecated loops, bare includes, Python 2
  **R** — Risk: shell injection, privilege escalation, insecure downloads
  **SEC** — Secrets: passwords, API keys, tokens (Gitleaks)
  **P** — Policy: module allow/deny, argument validation
